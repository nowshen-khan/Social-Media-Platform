# Social-Media-Platform
Social Media Platform

# Description
This is a social media platform where users can create accounts, post updates, like and comment on posts, and follow other users.

# Features
* User authentication and authorization
* Posting updates with text, images, and videos
* Liking and commenting on posts
* Following other users
* User profile page with timeline of posts

# Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Token)
- bcrypt (password hashing)
- nextjs

# API Endpoints
## User Endpoints
- Register a new user
- Login a user
- Get user profile
- Update user profile

## Post Endpoints
- Create a new post
- Get all posts
- Get a single post
- Update a post
- Delete a post

## Like and Comment Endpoints
- Like a post
- Comment on a post

## Follow Endpoints
- Follow a user
- Unfollow a user

# Authentication
Authentication is handled using JWT (JSON Web Token) for user authentication and authorization. Users are required to register and log in to access the platform.

# User Profile
User profile page displays a timeline of posts made by the user.

# Post Updates
Post updates can be created, edited, and deleted by users.

# Like and Comment on Posts
Users can like and comment on posts made by other users.

# Follow Other Users
Users can follow other users and view their timeline of posts.
